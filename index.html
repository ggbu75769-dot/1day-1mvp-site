출시 상태

엔지니어링 출시 준비 완료. 소스 공개 릴리스는 인간 승인 대기 중. Runtime API는 GitHub Pages에 호스팅하지 않음.

보안 경계

이 공개 GitHub Pages 사이트는 정적 페이지입니다. 백엔드, 런타임 토큰, 비공개 승인 기록, 애플리케이션 소스 코드를 노출하지 않습니다.

Pages에는 공개 출시 문구와 정적 에셋만 배포됩니다.
1Day-1MVP 공개 랜딩 페이지입니다. 메인 애플리케이션 저장소는 비공개로 유지되며, 인간 승인과 라이선스 결정이 완료되기 전에는 소스 공개 릴리스가 승인되지 않습니다.
